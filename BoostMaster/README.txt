BoostMaster Plugin for BakkesMod
--------------------------------

Build instructions:
- Requires BakkesMod SDK and dependencies.
- Build as x64 Release or Debug in Visual Studio 2019/2022 with C++17.
- Include thirdparty/json.hpp in your include path.

Features planned:
- Boost pad pathfinding (A* or Dijkstra)
- Boost usage tracking
- Training drill save/load
- HUD and settings UI

Usage:
- Load plugin in Free Play mode.
- Use CVars and commands to save/load drills, visualize paths.
